# Dry-run: локальные сборки

Команда (подставьте `hostName` из `flake.nix`, по умолчанию `nixos`):

```bash
nix build .#nixosConfigurations.nixos.config.system.build.toplevel --dry-run
```

или с фиксированным именем хоста из вашего `flake.nix`.

На целевой машине после изменений имеет смысл повторить `--dry-run` и смотреть строки «will be built».

# Оценка места (пример)

| Путь | Размер (пример) |
|------|-----------------|
| `/nix/store` | порядка сотен GiB при долгой жизни системы |
| `/run/current-system` | ~15–25 GiB |
| `~/.config` | зависит от приложений |
| `~/.cache` | часто десятки GiB — можно чистить |

Игры и тяжёлые медиа в таблицу не входят.
